# 开发日志 — 2026-07-11

## 项目汇总

### Ratio Tracker — 极简比率跟踪器 v1.0

**核心功能**：跟踪 HG/GC 铜金比 和 SOXX/QQQ 半导体强弱比，实时计算 + 走势展示。

**数据源**：Yahoo Finance v8 API（4 个 ticker：HG=F, GC=F, SOXX, QQQ）

**技术栈**：SwiftUI + SwiftData + WidgetKit + Charts

## 开发历程

### 第一阶段 — 宏观仪表盘（已废弃）
- 设计了 13 个宏观经济指标的完整仪表盘
- 实现了 FRED API + Yahoo API 双数据源
- 包含信号概览、统计面板、多指标对比
- 最终简化为比率跟踪器

### 第二阶段 — 简化重构
- 删除 FRED 服务和 9 个非比率指标
- 从 39 个文件精简到 12 个文件
- 单窗口、两张卡片、两个比率

### 第三阶段 — iOS 移植
- 添加 iOS Widget Extension target
- 解决平台兼容：`#if os(macOS)` 条件编译
- 添加 `import Combine`（iOS 显式导入）
- 添加 `containerBackground`（WidgetKit iOS 17+ 要求）
- Info.plist 配置 UILaunchScreen（消除黑边）
- 创建 WidgetInfo.plist（含 NSExtension 声明）
- 配置 AppIcon 和 Asset Catalog

### 关键问题解决记录

| 问题 | 解决方案 |
|------|----------|
| pbxproj 手工编辑损坏 | 多次从 macOS 项目恢复，最终用生成脚本一步创建完整项目 |
| iOS 黑边 | Info.plist 添加 UILaunchScreen 字典 |
| Widget NSExtension 缺失 | 创建独立 WidgetInfo.plist |
| Widget containerBackground 报错 | 添加 `.containerBackground(.fill.tertiary, for: .widget)` |
| Combine/ObservableObject 编译错误 | 添加 `import Combine` |
| App 图标不显示 | Asset Catalog + ASSETCATALOG_COMPILER_APPICON_NAME + Resources Build Phase |
| Widget 调试报错 | Scheme 选 iOS App 而非 Widget |
